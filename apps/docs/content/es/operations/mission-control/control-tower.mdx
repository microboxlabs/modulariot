# Control Tower

The primary operational dashboard for fleet visibility and management.

## Overview

Control Tower provides a real-time view of your entire fleet, combining map visualization, symptom tracking, and asset management in a single interface.

## Main Components

### Fleet Map
Real-time asset positions displayed on an interactive map:
- Live location updates
- Status indicators
- Clustering for dense areas
- Multiple map views

### Symptom Panel
Active symptoms requiring attention:
- Count by severity
- Recent detections
- Quick actions

### Asset List
Tabular view of all assets:
- Sortable columns
- Quick filters
- Bulk actions

## Cómo Acceder a la Torre de Control

Se accede a la Torre de Control a través de la sección **Torre de Control** en la barra lateral de la aplicación, bajo la entrada Vista Geográfica. La barra lateral también muestra contadores en vivo para cada sub-sección antes de abrirlas — el contador de Vista Geográfica muestra el número de activos actualmente rastreados en el mapa, y el contador de Síntomas muestra el total de síntomas activos.

Para una referencia completa de las secciones de la barra lateral, contadores de insignias y control de acceso por grupo de usuario, ver [Navegación de la Aplicación](/operations/mission-control/navigation).
