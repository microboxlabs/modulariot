# Release Notes - ModularIoT v1.1

## 1. Mejoras y Nuevas Funcionalidades

### 1.1 Optimización de la API y Visualización de Mapas
- Actualización del endpoint de la API para mejorar la confiabilidad en la obtención de datos de mapas
- Implementación de filtrado avanzado con opciones de alternancia y borrado dinámico
- Mejora en la gestión del estado para visualizaciones en tiempo real
- Ampliación de la información de activos en mapas

### 1.2 Integraciones para Monitoreo en Tiempo Real
- Nuevos endpoints de API para obtención de síntomas UCI y posiciones en mapas
- Implementación de popups en mapas con detalles de activos y timestamps
- Mensajes de error visibles para mejor retroalimentación
- Nuevo modelo de transmisión de datos de posición en tiempo real para seguimiento de viajes

### 1.3 Experiencia de Usuario Mejorada
- Soporte de identificador de viaje en vista de mapas
- Nueva sección expandible para información detallada del viaje en timeline
- Actualización de indicadores de estado de síntomas
- Implementación de configuraciones multilingües (inglés/español)
- Optimización de navegación móvil con menú dinámico

### 1.4 Mejoras en Carga y Renderización
- Nuevas pantallas de carga con animaciones de esqueleto
- Mayor estabilidad en visualización con manejo seguro de datos faltantes
- Optimización en recuperación de grandes volúmenes de datos

## 2. Refactorización y Optimización

- Consolidación de lógica para obtención de datos de mapas y síntomas UCI
- Mejora en estructura y seguridad del manejo de datos
- Refinamiento en definición de datos de componentes
- Optimización de estados de carga
- Actualización de componentes de mapeo con versiones optimizadas
- Ajustes en configuración inicial de zoom y posicionamiento en mapas

## 3. Mejoras de Diseño

- Refinamiento de elementos visuales (tipografías, márgenes y espaciado)
- Mayor visibilidad y accesibilidad en botones de cierre
- Efectos visuales mejorados en filtros
- Rediseño del menú lateral
- Actualización de tarjetas de síntomas con títulos dinámicos e iconografía
- Implementación de efectos de desplazamiento suaves
- Optimización en diseño de interfaz y comportamiento visual

## 4. Documentación y Seguridad

- Ampliación de localización para tratamiento de síntomas en inglés y español
- Mejora en seguridad de tipos para datos de internacionalización
- Limpieza de logs de depuración en configuración de usuario

## 5. Coordinador de Viajes

- Implementación de detección automática de llegadas de viajes mediante integración con Alerce
- Sistema de monitoreo en tiempo real para seguimiento de viajes
- Optimización en flujo de trabajo para validación de contratistas y transportes
