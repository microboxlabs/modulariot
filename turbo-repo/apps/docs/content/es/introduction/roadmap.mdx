# Roadmap & Status

Current development status and planned enhancements.

## Current Status

ModularIoT is in active development with core capabilities available.

### Available Now
- Telemetry ingestion pipeline
- Canonical metrics processing
- Basic symptom detection
- Control Tower dashboard

### In Development
- Advanced symptom patterns
- SDK libraries (JavaScript, Python, Java)
- Enhanced notification channels
- Historical analytics

### Planned
- Predictive analytics
- Machine learning integration
- Custom dashboard builder
- Mobile applications

## Release Cadence

- **Minor releases** - Bi-weekly with bug fixes and improvements
- **Major releases** - Quarterly with new features
