# What is ModularIoT?

ModularIoT is a real-time operational intelligence platform designed to transform raw telemetry from IoT devices into actionable insights for fleet and asset management.

## Beyond Simple Data Collection

Traditional telemetry platforms focus on collecting and storing data. ModularIoT goes further by:

- **Normalizing** diverse device data into canonical formats
- **Detecting** operational conditions (symptoms) in real-time
- **Enabling** structured operational responses through Control Tower

## Platform vs API

ModularIoT is not just an API for storing telemetry. It's an operational platform that:

- Processes millions of data points per second
- Maintains stateful symptom detection
- Provides operator workflows for incident response
- Integrates with existing systems through flexible interfaces

## Who Uses ModularIoT?

- **Fleet operators** managing vehicle telemetry
- **Asset managers** tracking equipment health
- **Operations teams** responding to real-time conditions
- **Developers** integrating telemetry into applications
