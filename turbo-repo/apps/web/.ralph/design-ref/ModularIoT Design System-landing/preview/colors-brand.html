Semantic usage

OK / success
Neutral Nevada 600 icon. No green. Success is quiet.
Warning
Neutral Nevada 600 icon, same visual weight as OK. Do not reach for yellow.
Error
Selective Yellow 600. The one thing that pulls the eye.
Critical
Blaze Orange 700. Destructive action, código negro, escalation.

Rule of thumb: one action at a time. Color is reserved for the single thing the operator should do next. Everything else — status, metadata, success — is Nevada gray.
