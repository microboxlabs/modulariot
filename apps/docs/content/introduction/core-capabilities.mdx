# Core Capabilities

ModularIoT provides four foundational capabilities that work together to deliver operational intelligence.

## Telemetry Ingestion

High-throughput, multi-protocol data intake:

- Support for common IoT protocols
- Device-specific driver framework
- Automatic data normalization
- Reliable delivery guarantees

## Metric Processing

Transform raw telemetry into canonical measurements:

- Standardized metric schemas
- Unit normalization
- Quality validation
- Enrichment with context

## Symptom Detection

Real-time identification of operational conditions:

- Stateful symptom tracking
- Severity classification
- Automatic escalation
- Historical analysis

## Operational Workflow

Tools for responding to detected conditions:

- Control Tower dashboard
- Symptom worklists
- Operator action tracking
- Audit trail
