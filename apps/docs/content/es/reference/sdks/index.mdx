# SDKs

Bibliotecas cliente oficiales para integrar con ModularIoT en diferentes lenguajes de programación.

## SDKs disponibles

| Paquete | Lenguaje | Estado |
|---------|----------|--------|
| [`@microboxlabs/miot-calendar-client`](/es/reference/sdks/miot-calendar-client) | TypeScript | **Disponible** |
| `@modulariot/sdk` | JavaScript | Próximamente |
| `modulariot` | Python | Próximamente |
| `io.modulariot:sdk` | Java | Próximamente |

## Inicio rápido

Cada SDK proporciona una interfaz consistente para:

- **Autenticación** - Soporte para API key y OAuth 2.0
- **Ingesta de telemetría** - Envío de métricas y eventos
- **Consulta de síntomas** - Lectura del estado de síntomas
- **Manejo de webhooks** - Procesamiento de callbacks de la plataforma

## Documentación de SDKs

Selecciona tu lenguaje en la barra lateral para ver la referencia de API detallada y ejemplos de uso.

## API REST

Si no hay un SDK disponible para tu lenguaje, puedes integrar directamente con la API REST. Consulta la sección de **Referencia de API** para documentación completa de endpoints.
