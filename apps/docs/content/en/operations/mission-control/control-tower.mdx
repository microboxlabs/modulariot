# Control Tower

The primary operational dashboard for fleet visibility and management.

## Overview

Control Tower provides a real-time view of your entire fleet, combining map visualization, symptom tracking, and asset management in a single interface.

## Main Components

### Fleet Map
Real-time asset positions displayed on an interactive map:
- Live location updates
- Status indicators
- Clustering for dense areas
- Multiple map views

### Symptom Panel
Active symptoms requiring attention:
- Count by severity
- Recent detections
- Quick actions

### Asset List
Tabular view of all assets:
- Sortable columns
- Quick filters
- Bulk actions

## Accessing Control Tower

Control Tower is reached via the **Control Tower** section in the application sidebar, under the Geographic View entry. The sidebar also shows live badge counts for each sub-section before you open them — the Geographic View badge shows the number of assets currently tracked on the map, and the Symptoms badge shows the total count of active symptoms.

For a full reference of sidebar sections, badge counts, and access control by user group, see [Application Navigation](/operations/mission-control/navigation).

## Navigation

### Fleet View
Overview of all assets and symptoms.

### Asset Detail
Deep dive into single asset:
- Current status
- Telemetry history
- Active symptoms

### Symptom Detail
Investigation view for symptoms:
- Full context
- Related data
- Action history
- Resolution workflow
